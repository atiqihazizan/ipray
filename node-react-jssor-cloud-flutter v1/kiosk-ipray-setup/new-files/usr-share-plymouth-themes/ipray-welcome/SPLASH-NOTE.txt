Fail splash.png tidak disertakan di repo (imej binari).

Di Pi, cipta atau salin imej ke:
  /usr/share/plymouth/themes/ipray-welcome/splash.png

Contoh:
  - Salin dari tema pix: cp /usr/share/plymouth/themes/pix/splash.png /usr/share/plymouth/themes/ipray-welcome/
  - Atau letakkan imej welcome/logo custom anda (contoh 1920x1080 PNG).

Setelah splash.png wujud, jalankan: sudo update-initramfs -u
