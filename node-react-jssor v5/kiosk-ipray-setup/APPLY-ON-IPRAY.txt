# Cara apply perubahan kiosk + config di ipray (kurangkan blink)

Apabila SSH ke ipray berjaya, jalankan dari folder projek (node-react-jssor) di mesin anda:

--- 1) Backup di ipray ---
ssh ipray "mkdir -p /home/ipray/backup/$(date +%Y-%m-%d)"
ssh ipray "cp /home/ipray/kiosk.sh /home/ipray/backup/$(date +%Y-%m-%d)/kiosk\ copy.sh1.bak"
ssh ipray "sudo cp /boot/firmware/config.txt /home/ipray/backup/$(date +%Y-%m-%d)/config\ copy.txt1.bak"

--- 2) Copy kiosk.sh (dari folder ini) ke ipray ---
scp kiosk-ipray-setup/kiosk-ipray.sh ipray:/home/ipray/kiosk.sh
ssh ipray "chmod +x /home/ipray/kiosk.sh"

--- 3) Copy config.txt (perlu sudo di ipray) ---
scp kiosk-ipray-setup/config-ipray.txt ipray:/tmp/config.txt
ssh ipray "sudo cp /tmp/config.txt /boot/firmware/config.txt"

--- 4) Reboot ipray (supaya config.txt gpu_mem berkesan) ---
ssh ipray "sudo reboot"

Selepas reboot, kiosk akan guna kiosk.sh baru. Jika autostart guna script lain, pastikan anda ganti kandungan script autostart dengan yang sama flag seperti dalam kiosk-ipray.sh.
