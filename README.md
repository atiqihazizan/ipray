# iPray

A C++ SFML application for prayer times and related functionality.

## Prerequisites
- C++ compiler
- SFML library
- Make

## Building
```bash
make
```

## Running
```bash
./ipray
```

## Project Structure
- `components/` - Core components
- `fonts/` - Font resources
- `images/` - Image assets
- `include/` - Header files
- `lib/` - Library files
- `src/` - Source files
