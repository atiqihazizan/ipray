Override Kuliah: Fail ini menyimpan peraturan tarikh + type kuliah untuk batal atau ganti paparan (cth. Solat Tarawih, Hari Raya). Tarikh yang match akan dipaparkan sebagai "BATAL" atau diganti dengan catatan (ikut flag).

Format disokong:

  • Tarikh tunggal (Masihi): DD-MM-YYYY|type|notes
  • Range Masihi: tahun?|bulan|type|hari|flag|catatan (tahun kosong = tahun semasa)
  • Tarikh Hijri: hijri|tahun|bulan|hari|type|flag|catatan (wajib). Pilihan papar di Pengumuman: tambah |1|tajuk|tempat|jemputan (flag 1 = papar di slide Pengumuman).

Type: ks, km, kd, kk. Ganti paparan (1): slot kuliah diganti dengan catatan. Paparkan di announcement (1): event akan muncul dalam slide Pengumuman dengan format PENGUMUMAN|tajuk|note|tarikh|tempat|jemputan.
